# TODO アプリ

このアプリは Flutter を使用した TODO アプリです。
Flutter の開発で必要な基本知識の習得に役立てるリポジトリです。

## このアプリの特徴

本アプリでは以下の特徴を持っています。

- メモの追加、編集、削除、取得
- アーカイブ機能（ゴミ箱）
- メモのページング機能
- ゴミ箱に存在しているメモの自動削除機能
- SnackBar の表示、取消機能
